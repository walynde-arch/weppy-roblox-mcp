# Roblox Documentation Sources

When using Weppy MCP tools, look up Roblox engine APIs, classes, or patterns through these sources.

## llms.txt — LLM-optimized indexes

- **Engine API Reference**: https://create.roblox.com/docs/reference/engine/llms.txt
  Index of 654+ classes, enums, datatypes, globals, and libraries.
  Fetch specific pages like `/docs/reference/engine/classes/Part.md`.

- **Creator Documentation**: https://create.roblox.com/docs/llms.txt
  Full creator docs index — guides for scripting, UI, lighting, terrain, audio, etc.

- **Open Cloud API**: https://create.roblox.com/docs/cloud/llms.txt
  REST API reference for external server-side integrations (asset upload, data stores).

## How to use

1. Read the llms.txt index to find the class or topic
2. Fetch the specific `.md` page for full details (properties, methods, events, samples)
3. Use engine API docs for Luau code inside Roblox
4. Use Open Cloud docs for REST API calls from external tools
